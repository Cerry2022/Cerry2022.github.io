VS Code 插件 markdownlint 提示 “no-hard-tabs” ​

VS Code 启用 markdownlint 插件编写 md 文件的时候，发现提示 MD010/no-hard-tabs：Hard tabs：

好奇调查了一下：

所谓的 hard tab 是指 制表符，ASCII 码为 9，C语言中转义序列表示为 \t。

与 hard tab 相对应的是 soft tab，是由 2个 或 4个 空格组成。
