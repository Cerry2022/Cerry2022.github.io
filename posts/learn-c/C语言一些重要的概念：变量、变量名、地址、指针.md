## 3、指针
### 指针变量
```c
int   a = 112, b = -1;
float c = 3.14;
int   *d = &a;
float *e = &c;
```

| 100 | 104 | 108  | 112 | 116 |
| --- | --- | ---- | --- | --- |
| a   | b   | c    | d   | e   |
| 112 | -1  | 3.14 | 100 | 108 |

值：
- a : 112
- b : -1
- c : 3.14
- d: 100
- e: 108

### 间接访问
通过一个指针访问他所指向的地址的过程称为**间接访问**或**解引用指针**：
使用单目操作符 `*`
```c
int   f = *d;
```

### 未初始化和非法的指针
一个极为常见的错误：
```c
int *a;
...
*a = 12;
```
无法预测 12 会存储在哪个地方，
声明一个指向整型的指针不会“创建”用于储存整型的内存空间。
一般情况，会指向一个非法地址：
	在UNIX上会导致“内存错误”（memory fault）
	在Windows上会导致“保护性异常”（general protection exception）
最严重的情况，可能指向了一个合法的地址：
	位于那个位置的值被修改，这种错误难以捕捉，导致严重后果。
